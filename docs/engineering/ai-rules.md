# Fynora — AI Rules

## Objetivo

Garantir que a IA mantenha consistência no código.

---

## Regras

* seguir arquitetura definida
* usar TypeScript
* respeitar estrutura de pastas
* não criar padrões novos

---

## Backend

* NestJS obrigatório
* usar DTO
* separar camadas

---

## Banco

* usar Prisma
* respeitar schema
* usar migrations

---

## Multi-tenant

* sempre usar company_id
* nunca acessar dados sem validação
