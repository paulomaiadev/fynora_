# Fynora — Product Requirements Document (PRD)

## Objetivo

Permitir que MEIs gerenciem seu negócio de forma simples, profissional e baseada em dados.

---

## Funcionalidades do MVP

### Dashboard

* faturamento do mês
* despesas do mês
* lucro
* gráfico de desempenho

---

### Receitas

* registrar entrada
* valor
* data
* descrição
* cliente (opcional)

---

### Despesas

* registrar saída
* valor
* categoria
* data
* descrição

---

### Clientes

* cadastrar cliente
* histórico de receitas

---

### Projetos

* criar projeto
* associar cliente
* valor
* status

---

### Autenticação

* cadastro
* login

---

### IA (Gestor Inteligente)

* análise de dados financeiros
* geração de insights
* alertas de risco

---

## Métricas de sucesso

* usuário registra dados
* usuário retorna ao sistema
* usuário entende seu lucro
