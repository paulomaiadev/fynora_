# Fynora

O sistema que transforma um MEI em gestor profissional.

---

## 📌 Sobre o projeto

Fynora é uma plataforma SaaS voltada para MEIs e pequenos empreendedores, com foco em gestão simples, intuitiva e profissional.

---

## 🚀 Funcionalidades

* gestão financeira (receitas e despesas)
* controle de clientes
* gestão de projetos
* dashboard com métricas
* insights com inteligência artificial

---

## 🧠 Objetivo

Permitir que qualquer empreendedor consiga gerenciar seu negócio de forma profissional, mesmo sem conhecimento técnico.

---

## 🏗️ Stack

Frontend:

* Next.js
* TypeScript
* TailwindCSS

Backend:

* NestJS
* Prisma
* PostgreSQL

---

## 📁 Estrutura

* `/docs` → documentação completa
* `/backend` → API
* `/frontend` → interface
* `/prisma` → banco de dados

---

## ⚙️ Como rodar (futuro)

```bash
# instalar dependências
npm install

# rodar backend
npm run start:dev

# rodar frontend
npm run dev
```
